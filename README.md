# 数字人情感陪伴系统 (Digital Human Companion)

这是一款基于大模型技术与图生视频能力构建的“数字人情感陪伴系统”。系统旨在为用户提供高度个性化、有记忆、有情绪反应的数字人物陪伴体验。用户可以自由创建心仪的数字角色（如老师、朋友或各类虚拟身份），并与其进行文字、语音以及动态视频的双向互动。

## 主要功能特性

*   **沉浸式双向互动**：支持文字聊天、语音语音朗读（基于大模型 TTS），为用户提供真实的交流体验。
*   **数字人动态视频生成**：集成阿里云万象图生视频接口（`wan2.6-i2v-flash`），能够在用户与数字人物交流时生成专属口型或微表情动作视频。
*   **智能长效记忆机制**：拥有长期记忆模块与情绪分析模块（支持自动记录“职业规划”、“情感状态”、“个人信息”等，甚至通过情绪分析给予关怀提醒），让数字人能够“记得”你们聊过什么。
*   **一键切换数字人**：用户可以创建多名不同性格、身份与基础设定（System Prompt）的数字人角色，自由在多个角色之间切换。
*   **响应式前端体验**：结合 Vue3 的动态 UI，支持粒子特效背景、浮窗情绪提示和丝滑动画。

## 技术栈架构

*   **前端**：Vue 3 + Vite + GSAP (动画)
*   **后端**：FastAPI (Python)
*   **数据库**：MongoDB (Motor 异步驱动)
*   **大模型服务**：阿里云百炼平台（DashScope - Qwen-Plus 语义模型、Wanx 万象视频生成模型）
*   **云存储服务**：阿里云 OSS（用于头像及动态视频存储）

## 快速运行说明

详细环境搭建步骤与前后端运行命令，请参考项目根目录下的 [运行说明文档.md](./运行说明文档.md)。

### 前端运行
```bash
cd frontend
npm install
npm run dev
```

### 后端运行
```bash
cd backend
# 激活虚拟环境后安装依赖
pip install -r requirements.txt
# 运行 FastAPI 后端
python run.py
```

### 环境变量与密钥说明
在运行后端之前，请务必在 `backend/` 目录下创建一个 `.env` 文件。
可以参考 `backend/.env.example` 中的模板，填入你个人的 `API_KEY` 及数据库配置，特别注意不要将包含真实密钥的 `.env` 文件意外提交至 GitHub！
