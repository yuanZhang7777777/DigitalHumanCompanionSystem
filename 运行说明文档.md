# 项目运行操作文档

本项目分为前端和后端两部分，分别独立运行。您需要打开两个终端（Terminal）窗口来分别启动前端和后端服务。

## 环境要求
- **后端**：需要安装 Python 3.8 或以上版本。
- **前端**：需要安装 Node.js (推荐 v16 以上版本) 和 npm。

---

## 1. 运行后端服务

后端使用 Python 编写，基于 FastAPI 框架。

### 操作步骤：
1. **打开一个终端窗口，进入后端目录**
   ```cmd
   cd backend
   ```

2. **激活虚拟环境（如果使用了虚拟环境）**
   *注意：如果您的项目没有使用虚拟环境，可以直接跳过此步。*
   ```cmd
   # Windows 系统：
   venv\Scripts\activate
   # 如果您的虚拟环境名称是 .venv，请使用 .venv\Scripts\activate
   ```

3. **安装依赖（首次运行或依赖有更新时需要执行）**
   ```cmd
   pip install -r requirements.txt
   ```

4. **启动后端服务**
   直接运行 `run.py` 脚本：
   ```cmd
   python run.py
   ```
   *启动成功后，后端服务默认会运行在 `http://127.0.0.1:8000` 或 `http://localhost:8000`。*

---

## 2. 运行前端服务

前端基于 Vue 3 + Vite 构建。

### 操作步骤：
1. **打开一个新的终端窗口，进入前端目录**
   在项目根目录下执行：
   ```cmd
   cd frontend
   ```

2. **安装前端依赖（首次运行或 `package.json` 有改变时需执行）**
   ```cmd
   npm install
   ```

3. **启动前端开发服务器**
   ```cmd
   npm run dev
   ```
   *启动成功后，终端会打印出本地访问地址（通常是 `http://127.0.0.1:5173` 或类似地址），在浏览器中打开该地址即可访问前端页面。*

---

## 总结
日常开发测试时，您需要在两个独立的命令窗口中分别保持 `python run.py` 和 `npm run dev` 运行状态，这样前端页面才能成功调用后端的提供的接口服务。
